# Пульс продаж

Локальный интерактивный дашборд для анализа ежедневных продаж из Excel или CSV. Данные обрабатываются на компьютере пользователя и не отправляются во внешние сервисы.

## Запуск

```powershell
python app.py
```

Откройте `http://127.0.0.1:8000`. При запуске доступен демо-набор; для проверки импорта загрузите `test_sales.xlsx`.

## Поддерживаемые поля

Обязательны дата и сумма продаж. Регион, магазин, продукт и идентификатор транзакции рекомендуются. Если заголовки отличаются, используйте «Настроить колонки».

## Возможности

- KPI: продажи, транзакции, средний чек, дни с продажами;
- фильтры по периоду, региону, магазину и продукту;
- динамика по дням, неделям или месяцам;
- рейтинги регионов, продуктов и магазинов;
- просмотр детализации и экспорт отфильтрованных данных в CSV;
- импорт `.xlsx`, `.xlsm` и `.csv` до 25 МБ.
